♻️ E-Waste Pickup Management System

A Django-based web application that enables users to schedule e-waste pickups, track assigned agents in real time, and manage the complete collection workflow efficiently.

---

📌 Overview

The E-Waste Pickup Management System is designed to simplify electronic waste collection by connecting customers, collection agents, and administrators through a centralized platform.

The system allows customers to request pickups, agents to manage assigned collections, and administrators to monitor overall operations. It also includes real-time GPS-based agent tracking and payment workflow management.

---

✨ Features

👤 Customer Module

- User Registration and Login
- Request E-Waste Pickup
- Track Pickup Status
- View Pickup History
- Live Agent Tracking on Map
- Payment Module

🚚 Agent Module

- Agent Login
- View Assigned Pickups
- Update Pickup Status
- OTP Verification
- Real-Time GPS Location Sharing
- Manage Pickup Requests

🛠️ Admin Module

- Dashboard Analytics
- Manage Users
- Manage Agents
- Assign Pickups
- Monitor Collections
- Settlement Management

📍 Tracking System

- Live GPS-Based Agent Tracking
- Customer Location Monitoring
- Real-Time Location Updates
- Interactive Map View
